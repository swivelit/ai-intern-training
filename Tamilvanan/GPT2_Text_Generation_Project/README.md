# Text Generation using GPT-2 (Small)

## Project Overview
This project fine-tunes the GPT-2 small model on a custom text dataset and generates text based on prompts.

## Features
- Fine-tune GPT-2 (small)
- Generate paragraphs from prompts
- Easy inference script

## Dataset
Use any free text corpus:
- Wikipedia subset
- News articles
- Books

## How to Run
```bash
pip install -r requirements.txt
python train.py
python inference.py
```
