
# Handwritten Digit Recognition using CNN (MNIST)

## Project Overview
This project implements a **basic Convolutional Neural Network (CNN)** to recognize handwritten digits using the **MNIST dataset**.

## Features
- Plain CNN model (Keras + TensorFlow)
- >98% accuracy on test data
- Training & evaluation script
- Visualization of:
  - Convolutional filters
  - Feature map activations

## Dataset
- MNIST (loaded directly from Keras)

## How to Run
```bash
pip install -r requirements.txt
python train.py
```

## Output
- Accuracy & classification report
- Saved model
- Filter & activation visualizations
