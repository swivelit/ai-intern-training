
# Project 6: Question Answering using BERT (SQuAD v1.1)

This project fine-tunes a BERT/DistilBERT model for Question Answering using the SQuAD v1.1 dataset.

## Features
- Uses HuggingFace Transformers
- Supports DistilBERT for low-resource systems
- Command-line QA demo script

## Setup
```bash
pip install -r requirements.txt
```

## Train (lightweight)
```bash
python train.py
```

## Run QA Demo
```bash
python qa_cli.py
```
