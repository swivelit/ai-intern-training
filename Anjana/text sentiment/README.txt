
Project 5: Text Sentiment Analyzer (LSTM & GRU)

Dataset:
- IMDb Movie Reviews (Keras built-in)

Files:
- train_lstm.py : Train LSTM model
- train_gru.py  : Train GRU model
- demo.py       : Sentiment prediction demo (uses LSTM)
- lstm_model.h5 : Saved LSTM model
- gru_model.h5  : Saved GRU model

How to Run:
1. python train_lstm.py
2. python train_gru.py
3. python demo.py

Comparison:
- LSTM: Better long-term dependency learning
- GRU: Faster training, fewer parameters
