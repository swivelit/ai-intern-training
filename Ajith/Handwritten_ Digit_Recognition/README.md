
# Handwritten Digit Recognition using CNN (MNIST)

## Overview
This project implements a plain Convolutional Neural Network to classify handwritten digits using the MNIST dataset.

## Features
- CNN built using TensorFlow/Keras
- Achieves >98% accuracy
- Visualization of convolutional layer activations
- Saved trained model

## How to Run
```bash
pip install tensorflow matplotlib
python train.py
python visualize.py
```

## Output
- Trained model: mnist_cnn.h5
- Accuracy plot
- Activation visualizations
