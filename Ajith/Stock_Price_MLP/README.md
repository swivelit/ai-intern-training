# Stock Price Prediction using MLP

## Description
Predicts the next-day closing price of a stock using a Multi-Layer Perceptron regression model.

## Dataset
Yahoo Finance (downloaded automatically)

## Steps
1. Download stock price data
2. Preprocess & scale closing prices
3. Train MLP regression model
4. Predict next-day closing price
5. Plot predicted vs actual prices

## Run
pip install -r requirements.txt  
python src/train.py
