
# Email Spam Classifier (Classical Machine Learning)

## Project Overview
This project builds a **binary email spam classifier** using classical machine learning algorithms:
- Logistic Regression
- Support Vector Machine (SVM)
- k-Nearest Neighbors (k-NN)
- Naive Bayes

The goal is to classify emails as **Spam** or **Not Spam** using the **SpamBase dataset** from the UCI Machine Learning Repository.
