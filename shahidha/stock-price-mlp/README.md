
# Stock Price Prediction using MLP

## Project Description
This project predicts the **next-day closing stock price** using a **Multi-Layer Perceptron (MLP)** deep learning model.

## Dataset
- Source: Yahoo Finance (CSV)
- Example stock: AAPL

## Model
- Fully Connected Neural Network (MLP)
- Loss: Mean Squared Error
- Optimizer: Adam

## Evaluation
- MAE
- RMSE
- Actual vs Predicted price plot

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python model/mlp_model.py
