
RNN Project 5: Text Sentiment Analyzer (LSTM & GRU)

Dataset:
- IMDb Movie Reviews (Keras built-in)

Tasks:
- Build LSTM and GRU models
- Compare accuracy/performance
- Save trained models
- Run text sentiment demo

Files:
- train_models.py : trains LSTM & GRU, saves models, prints accuracy
- demo.py         : sentiment prediction using trained LSTM model

Steps to Run:
1. pip install tensorflow
2. python train_models.py
3. python demo.py
