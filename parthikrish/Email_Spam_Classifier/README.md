
# Email Spam Classifier

## Description
Binary classification project to detect spam emails using classical ML algorithms.

## Algorithms Used
- Logistic Regression
- Support Vector Machine (SVM)
- k-Nearest Neighbors (k-NN)
- Naive Bayes

## Dataset
SpamBase Dataset (UCI ML Repository)

Download link:
https://archive.ics.uci.edu/ml/datasets/spambase

Place `spambase.data` inside the `data/` folder.

## Outputs
- Accuracy comparison
- Confusion Matrix
- ROC Curve

## How to Run
1. Install requirements
2. Run notebook or src/train.py
