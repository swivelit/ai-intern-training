
# Loan Approval Prediction (Ensemble Models)

## Problem Statement
Predict whether a loan application should be approved using Decision Tree and Ensemble ML models.

## Dataset
Loan Prediction Dataset (Kaggle):
https://www.kaggle.com/datasets/ninzaami/loan-predication

Download the dataset and place `loan_data.csv` inside the `data/` folder.

## Models Used
- Decision Tree
- Random Forest
- XGBoost
- LightGBM
- CatBoost

## Evaluation Metric
- Accuracy
- Classification Report
- Confusion Matrix

Target accuracy: **>80%**

## How to Run
```bash
pip install -r requirements.txt
python main.py
```

## Output
- Trained models
- Feature importance plots
- Accuracy comparison
